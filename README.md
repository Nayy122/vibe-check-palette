# 🎨 Vibe Check: AI Color Palette Generator

[![Live Demo](https://img.shields.io/badge/demo-live-green.svg)](https://vibe-check-nay122.vercel.app/)

## 📌 About The Project
"Vibe Check" is a lightweight, AI-powered design utility that bridges the gap between conceptual feelings and technical design constraints. By typing in a mood or theme, the app uses an AI API to instantly generate a harmonious, five-color hex code palette that perfectly matches the vibe. 

## 🛠️ Tech Stack
* HTML5, CSS3, Vanilla JavaScript
* External AI API for natural language processing
* Deployed live via Vercel
